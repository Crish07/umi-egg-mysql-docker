# umi-egg-mysql-docker

## 简介

**umi 和 egg 结合的开源项目 持续更新中...**

- 2020-03-01 完成用户登陆验证及用户增删改查

## 启动方式

**启动 mysql 数据库**

```base
$ cd mysql
$ mkdir db

再查看 mysql 文件夹下 START.md 文件
```

**启动 egg**

```base
$ cd egg-server
$ npm i
$ npm run dev
```

**启动 umi**

```base
$ cd umi-antd-dva
$ npm i
$ npm start
```
